# 图像缩放、恢复与频域分析实验 (Assignment 3)

## 1. 实验项目简介
本实验通过对图像进行下采样与三种不同的插值恢复，研究数字图像处理中的信息丢失与重构质量。实验结合了空间域评估（MSE/PSNR）与频率域分析（FT/DCT），旨在深入理解采样定理与信号恢复的本质。

## 2. 实验内容与算法实现
### 2.1 空间域重构
* **下采样处理**：应用高斯平滑 (Gaussian Blur) 对原图进行预处理，有效防止了下采样过程中的混叠 (Aliasing) 效应。
* **插值算法对比**：
    * **最近邻插值 (Nearest)**：保留原始像素，但会导致边缘产生显著的马赛克与锯齿。
    * **双线性插值 (Bilinear)**：通过 4 个像素加权平滑过渡，减弱了锯齿，但细节丢失明显。
    * **双三次插值 (Bicubic)**：通过 16 个像素拟合，重构图像最平滑，在 PSNR 指标上表现最优。

### 2.2 频率域分析 (FT)

* **分析**：傅里叶变换频谱图显示，原图的高频细节（发丝、纹理）在下采样后丢失严重。恢复图像的频谱边缘呈现大面积衰减，证明了插值算法仅能恢复图像的低频轮廓，无法找回丢失的高频成分。

### 2.3 能量分布分析 (DCT)

* **能量聚集度**：实验统计了 DCT 系数左上角（低频区）的能量占比。
* **数据结果**：
    * 原图：51.0%
    * 最近邻：53.9%
    * 双线性：85.6%
    * 双三次：88.2%
* **物理解释**：DCT 能量向左上角聚集的程度与图像的“平滑程度”正相关。最近邻插值因引入虚假高频锯齿，能量聚集度最低；双三次插值因平滑效果最强，导致高频能量几乎消失，DCT 能量极度向低频区压缩。

## 3. 实验结论
1. **采样定理的影响**：高频信息的丢失是不可逆的。插值算法本质上是低通滤波器，无法弥补下采样过程中的信息缺失。
2. **算法优劣分析**：双三次插值在数值指标（MSE/PSNR）和视觉平滑度上均优于其他方法，但代价是抹除了更多的原始纹理细节。
3. **频率域的有效性**：通过对比 FT 与 DCT 的频率特征，能够直观定量地衡量图像重构的质量与信息丢失特征，相比单一的 MSE 指标更具理论深度。
