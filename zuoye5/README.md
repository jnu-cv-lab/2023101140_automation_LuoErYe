# 计算机视觉实验：图像几何变换与透视校正

## 1. 实验目的
本实验旨在通过实际编程操作，深入理解并对比计算机视觉中常见的三种图像几何变换：**相似变换（Similarity Transform）**、**仿射变换（Affine Transform）**以及**透视变换（Perspective Transform）**。
同时，将理论应用于实际，利用透视变换实现对倾斜拍摄的文档（A4纸）进行视角校正，观察并评估校正前后图像及文字的形变情况。

---

## 2. 实验环境
* **编程语言**：Python 3.x
* **核心库**：
  * `OpenCV` (`cv2`)：用于执行图像变换和交互
  * `NumPy`：用于矩阵计算和坐标系转换
  * `Matplotlib`：用于可视化展示

---

## 3. 实验内容与结果分析

### 3.1 几何变换的性质对比（实验一）
实验设计了一张包含**矩形（红）**、**圆（绿）**、**平行线（蓝）**和**垂直线（黑）**的测试图，分别对其施加相似变换、仿射变换和透视变换，观察每种变换对几何性质的影响。

**结果总结表：**

| 几何性质 | 相似变换 (Similarity) | 仿射变换 (Affine) | 透视变换 (Perspective) |
| :--- | :--- | :--- | :--- |
| **直线保持为直线** | ✅ 保持 | ✅ 保持 | ✅ 保持 |
| **平行线保持平行** | ✅ 保持 | ✅ 保持 | ❌ 不保持 (发生收敛/发散) |
| **垂直线保持垂直** | ✅ 保持 | ❌ 不保持 (夹角改变) | ❌ 不保持 (夹角改变) |
| **圆保持为圆** | ✅ 保持 | ❌ 变椭圆 | ❌ 一般变为复杂曲线/椭圆变形 |

**详细结果分析：**
1. **相似变换**：本质是平移、旋转和等比例缩放的组合。它**保留了所有的几何特征**，形状完全不变，只是大小和位置、角度发生了改变。
2. **仿射变换**：在相似变换的基础上加入了切变（Shear）和非等比例缩放。观察结果可知，**它能保持“平直性”（直线仍是直线）和“平行性”（平行线仍平行）**。但是，原图中的垂直交叉线不再垂直，正圆被拉伸成了椭圆，红色的矩形也变成了平行四边形。
3. **透视变换**：模拟人眼或相机的近大远小视觉效果。它**只能保持“平直性”**。原图中的蓝色平行线在变换后不再平行（有相交的趋势）；红色矩形变成了不规则的四边形（两组对边均不再平行）；圆和垂直关系的畸变最为严重。

### 3.2 真实场景的透视校正（实验二）
**实验任务**：拍摄一张桌面上带有文字和表格的A4纸（存在透视畸变），通过代码将其校正为标准的平面正视图像。

**校正方法与过程**：
1. **交互式取点**：编写 `OpenCV` 鼠标回调函数，手动依次点击原图中A4纸的四个角点（左上、右上、左下、右下）。
2. **设定目标平面**：基于A4纸的物理标准比例（210mm × 297mm），按比例放大设定目标图像的像素分辨率（如 630 × 891），确保输出图像不会被异常拉伸。
3. **计算与变换**：利用 `cv2.getPerspectiveTransform` 计算透视变换矩阵，再使用 `cv2.warpPerspective` 进行校正映射。

**结果观察**：
* **整体形状**：原本由于拍摄角度呈现上宽下窄（或倾斜的四边形）的A4纸边框，被成功拉伸还原成了标准的矩形。
* **文字与表格**：校正前，表格的横线和竖线随着空间透视发生了倾斜和聚集；**校正后，表格线条重新恢复为水平和垂直状态**。矫正的文字内容清晰可读，**消除了透视带来的“近大远小”形变**，主观视觉满意度高，达到了“文档扫描仪”般的效果。

---

## 4. 核心代码说明

本实验严格遵循要求，主要使用了以下核心 API：

* **`cv2.warpAffine`**：用于执行相似变换和仿射变换。结合 `cv2.getRotationMatrix2D` (求相似/旋转矩阵) 或 `cv2.getAffineTransform` (根据3对点求仿射矩阵) 使用。
* **`cv2.warpPerspective`**：用于执行透视变换。需要结合 `cv2.getPerspectiveTransform` (根据4对点求透视矩阵) 计算出 $3 \times 3$ 的变换矩阵。

## 5. 运行说明

1. **运行几何变换测试**：
   执行 `python script1.py`。程序会自动读取 `test_image.png`，进行计算，并生成 `composite_result.png` 以及弹出 `Matplotlib` 对比图。
2. **运行A4纸透视校正**：
   执行 `python script2.py`。
   * 终端会提示“请按照严格顺序点击四个角”。
   * 在弹出的图片窗口中，**依次点击左上、右上、左下、右下**四个角。
   * 点击完第4个点后，程序会自动校正并保存为 `A4_corrected_result.jpg`，同时弹出校正后的预览窗口。

## 6. 实验总结
通过本次实验，我不仅直观地看到了不同几何变换对图像的影响，还掌握了利用透视变换解决实际生活问题（如文档扫描校正）的方法。实验证明，**变换的不变量越少（从相似到透视），它能模拟的形变就越复杂**。透视变换虽然破坏了平行和垂直等诸多性质，但它最符合三维世界到二维平面的真实投影规律，是计算机视觉中极为重要的一环。
